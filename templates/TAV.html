TAV

Music Visualizer

Built in Processing (2.7), using minim and sDrop

I worked on TAV (The Awesome Visualizer, I couldn't think of any interesting name back then) before starting college. I had recently picked up Processing and I was inspired by classic visualization present in iTunes, WinAmp, and Windows Media Player, so I decided to give it a try.

The visualizer had a few major checkboxes to tick, which included ease of use, a nice UI, smooth framerates, and customizability. I managed to get most of them working however making them work in Processing was a challenge, since it doesn't support scaling, and UI, atleast it didn't when I was working with it. Still it was a good experience making this.

This visualizer can run on 1080p at 60fps on a 3rd gen i5 3337u, however requires a Java runtime
